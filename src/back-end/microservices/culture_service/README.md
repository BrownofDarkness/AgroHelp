## How to Run

- Install the dependency

  ```bash
  npm install --save --force
  ```

- Run the project

  ```bash
  npm run dev
  ```


- Run with Docker
  
  ```bash
  docker-compose up
  ```

- Api Doc available at `http://localhost:5000/api-docs`
